## 六、补充常用注意事项

- 版本号规范：统一使用 `v数字\.数字` 格式（如v0\.1、v0\.2），便于mike版本管理和识别；

- 忽略文件：可在项目根目录添加 `\.gitignore` 文件，忽略node\_modules、缓存、日志等无需提交的文件夹/文件；

- 端口占用：若8080端口被占用，可修改 `mike serve` 后的端口号，如 `0\.0\.0\.0:8081`。
